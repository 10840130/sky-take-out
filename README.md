# 蒼穹外賣練習
使用Spring Boot框架所建立的外賣訂購系統。

用戶端使用微信小程序開發，故僅完成管理端。

## 使用技術
- 後端框架
  
    SpringBoot (3.1.2)
  
    mybatis

- 資料庫
  
    MySql
  
    Redis

- 前端框架
  
    Vue
  
    Uniapp


## 項目搭建
啟動nginx

導入sql

上傳文件服務

需要在yml配置文件中，填入AWS金鑰資料

帳戶:

admin

123456
